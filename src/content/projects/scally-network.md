I had originally built this using Jekyll (by hand! Pre-AI!). Which honestly sucked. It looked
nice because I used a Jekyll theme, but getting the photos to work properly, getting different types of articles working, etc.
was all such a pain.

So I basically just had Codex revamp using Astro instead of Jekyll. It's another static site
builder, but particularly Codex was able to build the different article types and gallery features
really quickly.

I still get to write my markdown files as my new posts, and I still have to deal with the
(extremely annoying) static site workflow. But it's definitely way better.
